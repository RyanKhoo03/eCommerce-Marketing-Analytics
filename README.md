# eCommerce Marketing Analytics Dashboard

## Objective
- Analyse eCommerce data to identify trends & seasonal fluctuations across different products & demographics
- Identify segments with lower conversion and give recommedations to improve conversion rate

## Tools
- Google Analytics 4 (GA4)
- Google Looker Studio

## Visualisation & Analysis
- Built interactive dashboards to track user behaviour, traffic sources, and purchase patterns  
- Analysed device-level performance (mobile vs desktop)  
- Conducted conversion funnel analysis from product view → add to cart → purchase  
- Explored revenue contribution across product categories  

## Key Insights
- Twice the number of mobile users compared to desktop
- 75% of total sales came from tournament related fees
- Conversion funnel analyses:
  - 21.4% of viewed items were added to cart
  - 6.9% of viewed items were purchased
- Significant drop-off observed at the checkout stage, particularly for mobile users

 
# Recommendations
- Optimise mobile checkout by simplifying forms to reduce checkout friction to boost mobile conversion rate.
- Introduce cart abandonment emails using GA4 & Shopify triggers to address low conversion
- Diversify revenue streams to reduce reliance on tournament-related fees during off season.
